```
Shape GetOriginShape() const
```

无。

## 返回值<a name="section1598675141718"></a>

[Shape](Shape-176.md)类型，TensorDesc描述的originShape。

## 异常处理<a name="section67066196172"></a>

无。

## 约束说明<a name="section244552231716"></a>

无。
