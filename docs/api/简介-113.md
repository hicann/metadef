# 简介<a name="ZH-CN_TOPIC_0000002486420694"></a>

TensorV2是张量管理类，用于存储和操作张量数据。该类记录了张量的形状、步长、偏移、数据格式、数据类型、存放位置，提供了数据地址访问、数据大小计算等接口。相对于Tensor类，TensorV2增加携带了步长和偏移信息。

>![](public_sys-resources/icon-note.gif) **说明：** 
>该接口为预留接口，为后续的功能做保留，当前版本暂不支持。

#include <tensor.h>
