# 简介<a name="ZH-CN_TOPIC_0000002343617953"></a>

InferFormatContext继承自ExtendedKernelContext，是一个用于Format推导的上下文类。该类的主要作用是在推导算子输出Format的过程中，提供必要的输入输出Format、输入输出Shape和输入Tensor访问接口。InferFormatContext是Format推导函数的入参，Format推导函数的相关解释请参考[InferFormat](InferFormat.md)。
