# 构造函数<a name="ZH-CN_TOPIC_0000002078447885"></a>

## 函数功能<a name="zh-cn_topic_0000001635304933_zh-cn_topic_0000001339266220_section36583473819"></a>

创建一个补维规则。

## 函数原型<a name="zh-cn_topic_0000001635304933_zh-cn_topic_0000001339266220_section13230182415108"></a>

-   构造的实例中，补维规则（mask\_）以及补维后的维度（size\_）均为0

    ```
    ExpandDimsType() : size_(0U), mask_(0U)
    ```

-   通过字符串创建一个补维规则

    ```
    explicit ExpandDimsType(const ge::char_t *const expand_dims_type)
    ```

-   通过int64\_t位域定义创建一个补维规则

    ```
    explicit ExpandDimsType(const int64_t reshape_type_mask)
    ```
