# GetData<a name="ZH-CN_TOPIC_0000001265399790"></a>

## 函数功能<a name="section230999"></a>

获取Tensor中的数据。

const uint8\_t\* GetData\(\) const返回的数据不可修改，uint8\_t\* GetData\(\)返回的数据可修改。

## 函数原型<a name="section13787192971720"></a>

```
const uint8_t *GetData() const
uint8_t *GetData()
```

## 参数说明<a name="section2078992"></a>

无。

## 返回值<a name="section18710936"></a>

Tensor中所存放的数据。

## 异常处理<a name="section34180703"></a>

无。

## 约束说明<a name="section39190879"></a>

无。
