# 构造函数<a name="ZH-CN_TOPIC_0000002078486245"></a>

## 函数功能<a name="zh-cn_topic_0000001936566892_section36583473819"></a>

CompileTimeTensorDesc类用于描述编译时的Tensor描述信息，包含dtype信息以及format信息。由于编译时无法确定shape，因此不包含shape信息。该函数为CompileTimeTensorDesc类的构造函数。

## 函数原型<a name="zh-cn_topic_0000001936566892_section13230182415108"></a>

```
CompileTimeTensorDesc()
```

## 参数说明<a name="zh-cn_topic_0000001936566892_section75395119104"></a>

无。

## 返回值说明<a name="zh-cn_topic_0000001936566892_section25791320141317"></a>

返回一个CompileTimeTensorDesc对象。

## 约束说明<a name="zh-cn_topic_0000001936566892_section19165124931511"></a>

无。
