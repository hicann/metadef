# GetSize<a name="ZH-CN_TOPIC_0000001265240874"></a>

## 函数功能<a name="section46831934202919"></a>

获取Tensor中的数据的大小。

## 函数原型<a name="section186399319293"></a>

```
size_t GetSize() const
```

## 参数说明<a name="section15860103919294"></a>

无。

## 返回值<a name="section1490114312914"></a>

Tensor中存放的数据的大小，单位为字节。

## 异常处理<a name="section16219204732913"></a>

无。

## 约束说明<a name="section6477115015296"></a>

无。
