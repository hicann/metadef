# 构造函数<a name="ZH-CN_TOPIC_0000002042368750"></a>

## 函数功能<a name="zh-cn_topic_0000001584945788_zh-cn_topic_0000001339250874_section36583473819"></a>

构造一个格式，格式包括原始格式、运行时格式、补维规则。

## 函数原型<a name="zh-cn_topic_0000001584945788_zh-cn_topic_0000001339250874_section13230182415108"></a>

```
StorageFormat()
StorageFormat(const ge::Format origin_format, const ge::Format storage_format, const ExpandDimsType &expand_dims_type)
```
