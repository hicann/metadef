# SetOriginFormat<a name="ZH-CN_TOPIC_0000001312720985"></a>

## 函数功能<a name="section1189413510313"></a>

向TensorDesc中设置Tensor的原始Format。

该Format是指原始网络模型的Format。

## 函数原型<a name="section6188134993114"></a>

```
void SetOriginFormat(Format origin_format)
```
