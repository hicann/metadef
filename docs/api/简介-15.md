# 简介<a name="ZH-CN_TOPIC_0000002446115270"></a>

ExeResGenerationContext继承自ExtendedKernelContext，用于生成执行资源的上下文类，主要用于在算子执行前，生成或配置执行所需的各种资源，包括：内存、异步执行流、同步资源、算子属性、输入输出张量的形状信息。
