# 简介<a name="ZH-CN_TOPIC_0000002454911353"></a>

OpInferShapeContextBuilder用于构建InferShapeContext。构造出的Context在算子Shape推导过程中作为入参，用于获取必要的算子输入输出数据。推导完成后，结果会被写回上下文中。

OpInferShapeContextBuilder继承关系图如下：

![](figures/zh-cn_image_0000002456205278.png)

使用步骤如下：

1.  构造ContextHolder。

    调用OpInferShapeContextBuilder接口，传入算子输入的Tensor和输出的TensorDesc描述等信息，最终调用Build\(\)接口构造一个ContextHolder<InferShapeContext\>对象。

2.  获取InferShapeContext

    通过ContextHolder调用GetContext接口，获取InferShapeContext。

3.  调用算子Shape推导函数InferShapeKernelFunc，将InferShapeContext作为函数入参，完成Shape推导，并写入推导的输出结果。
4.  通过InferShapeContext的GetOutputShape接口可以获取到推导的输出Shape。
5.  根据需要释放ContextHolder，释放完成后，此时由Build构造出来的InferShapeContext中的数据指针均无效。

>![](public_sys-resources/icon-note.gif) **说明：** 
>该类继承自OpContextBuilderBase类，在Build构建ContextHolder对象之前，需要调用OpContextBuilderBase的OpType、OpName、IONum或IOInstanceNum，以及AppendAttr接口，分别设置算子的类型、名称、输入输出个数、以及算子的属性。
