## 函数功能<a name="zh-cn_topic_0000001651063897_section8062655"></a>

## 函数原型<a name="zh-cn_topic_0000001651063897_section5455038"></a>

```
AutoMappingSubgraphIOIndexFunc GetAutoMappingSubgraphIOIndexFunc(domi::FrameworkType framework)
```
